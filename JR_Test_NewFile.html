🟢 Zone A — Google Sheet

👤 มะนาว · ⏳ ~1 สัปดาห์ · ทุกคนใช้เป็นแล้ว
  • ⬜ หน้าที่พนักงานแต่ละคน
  • ⬜ มอบหมายงาน / คิวรายวัน
  • ⬜ Gantt / คิวเขียนแบบ
  • ⬜ รายงานปัญหา + วิธีแก้
  • ⬜ ช่องติ๊ก "อ่านแล้ว" (ผูกค่าคอม)
  • ⬜ WFH เสาร์ → ติ๊กอ่านปัญหา
  • ⬜ Flow ที่ยังปรับบ่อย
⚠ สต๊อกห้ามทำที่นี่ — หลายคนตัดพร้อมกัน เลขชน

🔵 Zone B — HTML / Web

👤 พี่นัท + Claude · Supabase (ย้าย self-host ได้ทีหลัง)
  • ✅ ใบเสนอราคา QT (~80%) — เปิดมือถือได้
  • ⬜ ใบวางบิล BL
  • ⬜ ใบเสร็จรับเงิน INV
  • ⬜ ใบเบิกงวด
  • ⬜ ใบตัดประกอบผลิต
  • ⬜ ใบรับประกัน
  • ⬜ Backend API + ฐานข้อมูล
  • ⬜ Login (ป้องกันสูตรราคา)
  • ⬜ Export PDF ทุกเอกสาร
  • ⬜ ตัดสต๊อก (มือถือ)
  • ⬜ งานผลิต / ตัด / ประกอบ
📁 ไฟล์หลัก: JR_คิดราคา.html (~200KB) พร้อมต่อ backend

🟣 Zone C — Telegram / n8n

👤 น้องฟลุ๊ค · ✅ เชื่อมแล้ว · รันบน server ออฟฟิศ
  • ✅ บอทแจ้งคิวรายวัน
  • ✅ Sheet กรอกครบ → Telegram auto
  • ✅ แจ้งเซลล์ว่าวันนี้ไปไหน
  • ✅ รันบน server ออฟฟิศ
หมายเหตุ: ตัวแจ้งเตือนเท่านั้น ไม่เก็บข้อมูลหลัก ไม่ต้องเชื่อมเว็บ

300,000 – 700,000 4 งวด — 35% / 30% / เหลือ / 40k
700,000 ขึ้นไป 5 งวด — 25/25/25/เหลือ/40k
